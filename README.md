# Janaa Birthday Surprise

A personalized surprise webpage for Janaa featuring a cinematic intro, love story timeline, memory gallery, interactive reasons cards, and background music playlist support.

## Features

- Cinematic animated intro section
- Scroll-based reveal sections
- Interactive "29 Pieces Of My Heart" cards with modal display
- Photo gallery with lightbox navigation
- Background audio player that rotates through multiple songs without immediate repeats
- Confetti animations for birthday celebration
- Responsive layout with emotional storytelling

## How to Run

1. Open `index.html` in a browser, or serve the folder with a local server.
2. Click "Open Your Surprise" and use the music button to toggle audio playback.

## Project Structure

- `index.html` — main page markup
- `style.css` — page styling and animations
- `script.js` — interactive behavior, music playlist, gallery, and modal logic
- `assets/` — photos, music, and video assets used in the site

## Notes

- The background music player supports multiple tracks and prevents the same song from playing back-to-back until all available songs have played.
- The project is designed as a heartfelt birthday surprise and can be customized further with additional photos, messages, or songs.
